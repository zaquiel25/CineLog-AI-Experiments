# Auto-Update Session Notes Command

You are tasked with automatically updating and optimizing SESSION_NOTES.md for focused day-to-day context management in this CineLog-AI-Experiments project.

## Context
This is an ASP.NET Core movie logging application with sophisticated suggestion algorithms, TMDB API integration, and performance optimizations. The SESSION_NOTES.md file maintains cross-session context for development continuity and should focus on actionable, current information rather than comprehensive historical logs.

## Your Mission

1. **Analyze Current Session**: 
   - Review the current conversation to identify key accomplishments
   - Detect major code changes, technical decisions, and workflow improvements
   - Identify user preferences and patterns observed during the session
   - Note any blockers, issues encountered, or incomplete work

2. **Update Session Entry**:
   - Add/update current session entry with today's date (2025-MM-DD format)
   - Document session goals and what was actually accomplished
   - Record key technical decisions and their rationale
   - Note user preferences and workflow patterns observed
   - Establish clear next session priorities

3. **Optimize File Context**:
   - Remove outdated session entries (older than 3-4 days) unless they contain critical ongoing context
   - Keep essential architecture patterns and production status information
   - Maintain user preferences and workflow guidelines that affect current work
   - Preserve recent deployment information and current project status

## Change Description
$ARGUMENTS

Start by reading the current SESSION_NOTES.md file, analyze the current session conversation, then systematically update the file according to these specifications.
