# CineLog Claude Code Agents

Specialized agents for CineLog development. Use the Task tool to invoke them.

## Agent Roster

| Agent | Focus |
|-------|-------|
| `cinelog-movie-specialist` | Movie features, suggestions, CRUD, user data |
| `aspnet-feature-developer` | Full-stack MVC: controllers, views, services |
| `tmdb-api-expert` | TMDB API, caching, rate limiting |
| `ef-migration-manager` | Database schema, migrations, indexes |
| `performance-optimizer` | Caching, query optimization, memory |
| `performance-monitor` | Performance testing and validation |
| `docs-architect` | Documentation maintenance |
| `deployment-project-manager` | Production deployment coordination |
| `session-secretary` | Session notes and continuity |

## When to Use Agents

- **Complex, multi-domain tasks** — route to the appropriate specialist
- **Simple tasks** (typos, single-file edits) — work directly, no agent needed

## Common Workflows

- **New feature**: `cinelog-movie-specialist` + `aspnet-feature-developer` + `docs-architect`
- **Performance fix**: `performance-optimizer` + `performance-monitor`
- **Database change**: `ef-migration-manager` + `cinelog-movie-specialist`
- **Deployment**: `deployment-project-manager` + `performance-monitor`
